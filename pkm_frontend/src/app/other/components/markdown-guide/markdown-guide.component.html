Markdown Guide

Markdown is a lightweight markup language for formatting text. Joplin uses CommonMark with extensions like KaTeX for math and Mermaid for diagrams.

Text Formatting

  • Bold: **bold** → bold
  • Italic: *italic* → italic
  • Bold + Italic: ***bold italic*** → bold italic
  • Strikethrough: ~~text~~ → text

Headings

  • # H1
  • ## H2
  • ### H3 (and so on)

Lists

  • Unordered: - Item, * Item
  • Ordered: 1. First, 2. Second
  • Nested: 
    - Parent
      - Child
  • Checkbox: - [ ] Todo, - [x] Done

Links & Images

  • Link: [OpenAI](https://openai.com)
  • Image: ![Alt](image.jpg)

Code

  • Inline: \`code\` → code
  • Block: 
    ```js
    console.log('Hello');
    ```

Blockquotes

> This is a quote

Horizontal Rule

---

Tables

| Name | Age |
|------|-----|
| Tom  | 30  |
| Ana  | 25  |

KaTeX (Math)

  • Inline: $x = y + 2$ → \(x = y + 2\)
  • Block: 
    $$
    \\frac{a}{b} = c
    $$

Mermaid Diagrams

```mermaid
graph TD;
  A-->B;
  B-->C;
  C-->A;
```

Escaping Characters

Use \ to escape: \*, \#, etc.

HTML Inside Markdown

You can mix raw HTML:

<div class="box">Hello HTML</div>

Advanced Tips

  • Use <br> for manual line breaks
  • Use double spaces at end of line for soft line breaks
  • Nested formatting like **bold _italic_** works

For more, visit CommonMark Spec, KaTeX Docs, or Mermaid Docs.
